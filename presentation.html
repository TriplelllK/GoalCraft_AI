📌 Проблема

Постановка целей в крупных компаниях — хаотичный процесс:
  • Руководители формулируют цели размыто — «улучшить работу отдела», «повысить эффективность»
  • Нет объективной оценки качества целей — каждый HR судит субъективно
  • Цели не связаны со стратегией компании и внутренними нормативными документами (ВНД)
  • Нет каскадирования — цели руководителя не декомпозируются на подчинённых
  • Дублирование целей между сотрудниками и кварталами
  • Нет аналитики по подразделениям — непонятно, кто ставит хорошие цели, а кто нет
💡 Масштаб: ~450 сотрудников, ~8 подразделений, ~9 000 целей в год, 13 таблиц данных (§4.2)

💡 Наше решение

GoalCraft AI — гибридная AI-система (8 275 строк кода):
  • Автоматическая SMART-оценка любой цели за секунду (score 0.0 – 1.0, точность 99%)
  • AI-генерация 3–5 целей для конкретной должности с привязкой к ВНД
  • Переформулировка слабых целей в SMART-совместимые
  • OKR-маппинг — привязка целей к Objectives & Key Results
  • Каскадирование от руководителя ко всем подчинённым
  • Alert Manager — 6 типов уведомлений (critical / warning) для менеджеров и HR
  • Аналитика — дашборд + индекс зрелости + история изменений (F-15)

Гибридный подход: Детерминированные правила (313 строк) + LLM (GPT-4o-mini) + RAG Hybrid Search

🔍 RAG Pipeline: гибридный поиск по ВНД

Документ (ВНД) → Sentence chunking
(300 / 50) → N-gram hashing
(512d vectors) → BM25 index → Hybrid search
Формула гибридного скоринга:

score = cosine × 0.40 + BM25 × 0.35 + keyword_boost × 0.15 + doc_type_bonus × 0.10
КомпонентВесОписание
Cosine similarity0.40N-gram feature hashing (512d)
BM25 score0.35Term frequency × IDF (нормализовано)
Keyword boost0.15Точное совпадение ключевых слов
Doc type bonus0.10Приоритет: strategy > vnd > kpi
Преимущества нашего RAG:
  • ✅ Zero-dependency (без BERT/HuggingFace)
  • ✅ ~1000× быстрее BERT для русского
  • ✅ Offline-ready (нет внешних API)
  • ✅ Sentence-aware: не режет предложения
  • ✅ Dual-mode: Memory / Qdrant
