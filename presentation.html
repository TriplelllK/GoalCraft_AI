📌 Проблема

Постановка целей в крупных компаниях — хаотичный процесс:
  • Руководители формулируют цели размыто — «улучшить работу отдела», «повысить эффективность»
  • Нет объективной оценки качества целей — каждый HR судит субъективно
  • Цели не связаны со стратегией компании и внутренними нормативными документами (ВНД)
  • Нет каскадирования — цели руководителя не декомпозируются на подчинённых
  • Дублирование целей между сотрудниками и кварталами
  • Нет аналитики по подразделениям — непонятно, кто ставит хорошие цели, а кто нет
💡 Масштаб: ~450 сотрудников, ~8 подразделений, ~9 000 целей в год

💡 Наше решение

GoalCraft AI — гибридная AI-система:
  • Автоматическая SMART-оценка любой цели за секунду (score 0.0 – 1.0)
  • AI-генерация 3–5 целей для конкретной должности с привязкой к ВНД
  • Переформулировка слабых целей в SMART-совместимые
  • OKR-маппинг — привязка целей к Objectives & Key Results
  • Каскадирование от руководителя ко всем подчинённым
  • Аналитика — дашборд качества целей по подразделениям + индекс зрелости

Гибридный подход: Детерминированные правила + LLM (GPT-4o-mini) с graceful degradation

🧠 Гибридный подход

📐 Rule-based модуль

  • SMART-оценка по 5 критериям
  • 40+ глаголов действия
  • 60+ объектов HR-домена
  • Regex-паттерны для метрик и дат
  • 100% воспроизводимость
  • Всегда доступен

🤖 LLM модуль

  • Генерация целей по контексту
  • Переписывание слабых целей
  • OKR-маппинг (Objectives + KRs)
  • GPT-4o-mini (OpenAI API)
  • Graceful degradation:
    нет ключа → fallback на правила
✅ Система полностью работоспособна без LLM (rule-based) и с LLM (расширенный режим)

🧪 Как проверить своими данными

1️⃣ Загрузите свои ВНД / стратегии

Откройте http://localhost:8899/docs → POST /api/v1/documents/ingest → Try it out:

{ "documents": [{
    "doc_id": "MY-VND-01", "doc_type": "vnd",
    "title": "Ваш реальный ВНД или стратегия",
    "content": "Полный текст документа — скопируйте из Word/PDF...",
    "owner_department_id": "dep_hr",
    "department_scope": ["dep_hr", "dep_ops"],
    "keywords": ["ваши", "ключевые", "слова"]
}]}

📌 doc_type: vnd (ВНД), strategy (стратегия), kpi (KPI-фреймворк), policy (политика)

2️⃣ Оцените реальные цели сотрудников

POST /api/v1/goals/evaluate — вставьте реальную формулировку цели из вашей системы:

{ "employee_id": "emp_1", "goal_text": "Ваша реальная цель сотрудника...",
  "quarter": "Q2", "year": 2026 }

3️⃣ Сгенерируйте цели для должности

POST /api/v1/goals/generate — укажите фокус по вашим приоритетам:

{ "employee_id": "emp_1", "quarter": "Q2", "year": 2026,
  "count": 3, "focus": "ваш приоритет на квартал" }

4️⃣ Запустите автоматические тесты

cd backend_project
python qa/run_api_contract_tests.py    # 10 контрактных тестов
python qa/run_diagnostic_50.py         # 100 целей (50 bad + 50 good)
